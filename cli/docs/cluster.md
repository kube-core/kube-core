## `kube-core cluster setup config`

Generates cluster-config.yaml from core values.

```
USAGE
  $ kube-core cluster setup config

DESCRIPTION
  Generates cluster-config.yaml from core values.

EXAMPLES
  $ kube-core cluster setup config
```
