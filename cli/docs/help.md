## `kube-core help [COMMAND]`

Display help for kube-core.

```
USAGE
  $ kube-core help [COMMAND] [-n]

ARGUMENTS
  COMMAND  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for kube-core.
```
